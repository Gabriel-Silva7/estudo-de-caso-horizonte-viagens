# Horizonte Viagens

## Sobre o projeto

Este projeto foi desenvolvido como um estudo de caso para a disciplina de Banco de Dados. O objetivo foi criar e modelar um banco de dados para uma agência de viagens fictícia, aplicando conceitos de modelagem, normalização e relacionamentos entre tabelas.

## Tecnologias utilizadas

* MySQL
* MySQL Workbench
* SQL

## Estrutura do projeto

O repositório contém:

* Script SQL para criação do banco de dados;
* Modelo desenvolvido no MySQL Workbench;
* Arquivos com o processo de normalização;
* Diagrama do banco de dados.

## Como executar

1. Abra o MySQL Workbench.
2. Execute o arquivo `Script.sql`.
3. O banco de dados será criado automaticamente.

## O que foi desenvolvido

* Modelagem do banco de dados;
* Criação das tabelas;
* Definição de chaves primárias e estrangeiras;
* Relacionamentos entre as entidades;
* Processo de normalização até a 3ª Forma Normal (3FN).
